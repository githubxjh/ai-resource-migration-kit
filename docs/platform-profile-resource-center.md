# 致趣百川内容中心资源库画像

这份画像沉淀致趣百川「内容中心 > 资源库上传」场景的实操规则。平台名和模块名已确认可公开；公开版不暴露客户名称、后台域名、私有接口、token 或日志。

## 当前适配范围

- 平台模块：致趣百川内容中心。
- 操作位置：资源库上传。
- 支持资源类型：图片、视频、文档。
- 不包含：客户数据、后台 URL、登录信息、接口路径、cookie/token、真实资源 ID 和历史上传日志。

## 平台形态

- 资源库上传按图片、视频、文档三类资源组织。
- 资源名面向客户展示，不同资源类型之间也可能共享全局命名空间。
- 浏览器上传弹窗可能先暂存文件，只有最后确认后才真正上传。
- 一些平台会先把二进制文件传到对象存储，再通过单独后台调用创建资源中心记录。

## 实操限制

| 区域 | 实操规则 |
|---|---|
| 图片 | 项目画像未证明前，先把约 5 MB 当作常见限制。 |
| 视频 | 先把约 2 GB 当作硬上限；接近上限的视频按单条处理。 |
| 资源名 | 视频/资源名超过约 64 字符时，先视为风险。 |
| 文档 | 长 basename 可能生成后台记录但链接失效；必须做链接健康检查。 |

这些是规划默认值，不是所有平台的通用事实。私有后台适配器必须确认真实平台配置。

## 并发规则

- 下载、源清单解析、本地文件检查、哈希和离线报告生成可以并行。
- 同一个浏览器上传弹窗必须串行操作。
- 大视频应按单条批次处理。
- 超时或 token 过期后的后台未知状态，必须先核对，再进行下一次上传。

## 完成口径

对象存储成功、UI 成功提示、上传组件里出现文件名，或文件已被选中，都不等于完成。

任务只有在以下条件满足时才算完成：

- 后台资源中心存在可查询 ID 或等价稳定记录。
- 台账记录了后台名称、证据、操作人、时间和备注。

文档资源还可能需要通过链接健康检查。

## 公开安全命名

平台名和模块名可公开说明；不得公开后台域名、接口路径、客户名、资源 ID、源链接、登录信息、token、cookie 或历史日志。公开文档中使用“资源中心”“类对象存储上传”“创建资源记录”和“精确搜索”等通用说法。
