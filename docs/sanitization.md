# 脱敏指南

发布示例前，必须删除或替换：

- 客户名称。
- 真实后台域名、URL、API 路径和渠道名称。
- cookie、token、本地浏览器 profile、账号名。
- 真实资源 ID 和已上传链接。
- 真实源链接。
- 会识别出保密产品、项目、人员或活动的原始文件名。
- 包含请求头、堆栈或私有 ID 的原始日志。
- 真实项目中的本地绝对路径。

## 公开示例规则

- 使用虚构客户名。
- 使用虚构源清单行。
- 使用占位本地文件。
- 使用模拟后台 ID。
- 保留流程形状，不保留敏感事实。

## 安全写法

推荐：

```text
source_row_id=DOC-001
resource_name=产品介绍
backend_id=MOCK-1001
```

避免：

```text
source_row_id=<真实表格行>
resource_name=<真实客户资源名>
backend_id=<真实后台资源 ID>
```
