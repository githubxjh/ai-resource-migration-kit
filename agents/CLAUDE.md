# CLAUDE.md - 资源迁移代理入口

本文件供 Claude Code 使用，内容与 `AGENTS.md` 对齐。

## 第一性原则

目标不是让文件“看起来上传了”，而是建立任务级证据链，证明每个源清单行到底是完成、跳过、阻塞还是待处理。

## 必需行为

- 行动前读取 `HANDOFF.md`、`PROJECT_GUIDE.md`、`PROGRESS.md` 和 `ledger.csv`。
- 真实后台工作前读取 `docs/ai-onboarding.md`。
- 使用任何真实后台适配器前，先跑模拟流程。
- 优先使用小批次并逐批验证，不要做大批量黑盒执行。
- 真实后台动作前写明假设。
- 把后台写入视为高风险操作。
- 保持当前断点可在没有聊天记录的情况下恢复。

## 完成标准

任务只有在以下任一条件满足时才算完成：

- 已上传，并由后台 ID 或等价可查询证据验证。
- 已通过哈希/同源/人工确认被证明为重复，并记录为复用。
- 已明确标记为阻塞，并记录原因和下一步。

其他情况都保持为待处理或需复核。
